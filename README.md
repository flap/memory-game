# 🐱 Jogo da Memória - Gatinhos

Jogo da memória 4x4 com emojis de gatinhos. Encontre todos os 8 pares para vencer!

## 🎮 Como Jogar

1. Clique em uma carta para revelá-la
2. Clique em uma segunda carta para tentar encontrar o par
3. Se as cartas forem iguais, elas permanecem viradas
4. Se forem diferentes, elas voltam ao estado original
5. Encontre todos os pares para ganhar 🏆🎉
